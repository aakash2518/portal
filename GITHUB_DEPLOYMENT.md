## 🔒 Security Notes

### Environment Variables
The `.env` and `.env.local` files are NOT pushed to GitHub (they're in .gitignore).

When deploying or cloning, you need to create these files with:
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
